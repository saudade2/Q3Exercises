HTML

HyperText Markup Language

The HyperText Markup Language or HTML is a language created in 1990 by a scientist. It is used to communicate with others on the world wide web. You can communicate via websites or webpages.

CSS

Cascading Style Sheets

Cascading Style Sheets or CSS is a tool that is used to style and control the look of your webpage. CSS makes it easier to create and maintain your webpage or website. You can use 3 types of CSS, Inline CSS, Internal CSS, and External CSS.

Bootstrap

Bootstrap Version 4

Bootstrap Version 4 is a framework used to create Responsive Web Designs. Responsive Web Designs are used to make your websites look good on all devices. These devices can range from phones, tablets, laptops, and desktops.
